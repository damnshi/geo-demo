# GEO Growth Agent 可读概览

官网（正式产品）：https://geo.thinkive.com/  
所属公司：思迪信息（thinkive）  
品类：生成式引擎优化（GEO）/ 品牌 AI 友好度 / AI 可见度  

## 产品是什么

GEO Growth 是思迪信息旗下的品牌 AI 友好度平台。当用户向豆包、DeepSeek、Kimi、通义千问等 Agent 询问行业选型、产品推荐或品牌评价时，平台帮助品牌验证能否被看见、被理解、被引用并被推荐，并将缺口转化为可执行的优化任务。

**注意：** 此处 GEO 指 Generative Engine Optimization（生成式引擎优化），不是 geographic growth（地理增长）。

## 六大方法论

1. **评测** — 用真实 Agent 问答验证品牌能否被看见  
2. **诊断** — 拆解内容覆盖、实体清晰度、权威可信度、结构化数据、AI 可引用性  
3. **优化** — 生成 P0/P1/P2 任务与内容 Brief  
4. **发布** — 输出 llms.txt、FAQ、Schema、/ai/*.md 等 AI 资产  
5. **复测** — 同一组 Prompt 前后对比，验证优化效果  
6. **监控** — 追踪品类 Prompt、竞品占位与引用证据变化  

## 核心能力

- 多 Agent 基线评测（豆包、DeepSeek、Kimi、通义千问）  
- 八大维度站点健康诊断  
- 可见度竞争分析（品牌出现率、推荐入选率、引用证据率、Share of Voice）  
- Agent Markdown 资料包导出（/ai/*.md）  
- 同 Prompt 复测与效果归因  

## 适用行业

证券 IT、金融科技、银行资管、B2B SaaS、企业软件等需要在 AI 选型问答中争取可见度的品牌。
